Origm 推薦テンプレ・構造評価ダッシュボード

① 推薦テンプレUI（QuickAdd互換）

quickadd_template_ui_with_recommend.json
→ μテンションに基づくテンプレ推薦選択肢（QuickAdd向け）

② テンプレ即時ジャンプスクリプト

jump_trigger.html
→ 推薦テンプレを即時に開くためのJSスニペット

③ 推薦テンプレを fold_macro 形式で出力

fold_macro_from_recommend.yaml
→ μテンションと推薦関係に基づいて追加されたテンプレ群（fold_macro形式）

④ 推薦構造の可視化（Canvas形式）

  • テンション放射構造：中核テンプレからの放射的ベクトル
  • 推薦エッジ構造：fold間の推薦関係（矢印）

⑤ 構造クラスタ（テンション圏）

recommend_cluster_groups.yaml
→ μテンションに基づくテンプレ群のクラスタ分類

⑥ fold_macro補完（Canvas構造からの逆算）

fold_macro_from_canvas.yaml
→ Obsidian Canvas構造を解析して生成されたfold_macro形式テンプレ
